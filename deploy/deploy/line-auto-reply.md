## 四、設定步驟速查

1. **開通 LINE 官方帳號**：line.biz/zh-hant → 免費方案（每月 200 則主動推播訊息）
2. **設定歡迎訊息**：聊天 → 加入好友的歡迎訊息 → 貼上第一段
3. **設定關鍵字回覆**：自動回應訊息 → 逐一新增上方 7 組關鍵字
4. **上傳圖文選單**（選做）：用 Canva 做一張 2500x1686px 或 2500x843px 的圖
5. **放上 PDF 連結**：將 3 份 checklist PDF 部署到網站後，確認連結可存取
6. **在網站加入 LINE 連結**：替換 `https://lin.ee/YOUR_LINE_ID` 為你的實際 ID
